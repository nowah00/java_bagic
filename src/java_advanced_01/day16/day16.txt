트리 (Tree)
트리 : Node(열매, 데이터를 저장하고 자기 자신의 주소값을 보관하고 있는 저장단위), Branch(가지)를 이용해서, 사이클이 이루지 않도록 구성된 데이터 구조
    1. Node : 트리에서 데이터를 저장하는 기본 요소 (데이터와 다른 연결된 노드에 대한 Branch 정보 포함)
    2. RootNode : 트리에서 맨 위에 있는 시작 Node
    3. Level(Depth) : 최상위 Node 를 Level0 으로 하였을 때, 하위 Branch 로 연결된 노드의 깊이를 나타냄
    4. Parent Node(부모노드) : 어떤 노드의 위 Level 노드
    5. Child Node(자식노드) : 어떤 노드의 아래 Level 노드
    6. leaf Node(Terminal Node) : Child Node 가 하나도 없는 노드
    7. Sibling(Brother Node) : 현제 노드, 동일한 Parent Node 를 가진 노드


이진트리와 이진 탐색 트리 (Binary Search Tree)
    이진트리 : 노드의 최대 Branch 가 2인 트리
    이진탐색 트리 (BST) : 이진트리에 왼쪽 노드는 해당 노드보다 작은 값, 오른쪽 노드에는 해당 노드보다 큰값을 배치하는 조건이 있는 트리

    주요용도 : 데이터 검색(탐색)
    장점 : 탐색 속도를 개선할 수 있음
    단점 :

    이진트리와 정렬된 배열간의 탐색 비교
