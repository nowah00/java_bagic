1. 리소스 (resource)
   데이터를 제공하는 객체
   DAO(Data Access Object) => 데이터 처리 객체
   DTO(Data Transfer Object) => 데이터 전송 객체
   VO(Value Object) => 데이터 객체

   리소스를 사용하기 위서는 반드시 열어야 한다. => open()
   리소스 사용이 끝난 후에는 닫아야 한다. => close()

   예) 파일의 내용을 읽기 위해서는 파일 열어야 한다.
       다 읽은 다음에 파일 닫아야 다른 프로그램에서 사용할 수 있다.
       리소스를 상용하다가 예외가 발생할 수 있다.
       안전하게 닫는 것이 매우 중요하다. 리소스는 불안정 상태로 남아 있게되고
       이는 자칫 데이터의 손상을 야기할 수 있다.
